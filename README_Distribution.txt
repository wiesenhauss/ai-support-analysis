# WordPress.com Support Analyzer (macOS)

## Quick Start

1. **Download** the application bundle `WordPress_Support_Analyzer_vX.X.X.app`
2. **Set up your API key**:
   - Create a `.env` file in the same directory as the .app file
   - Add your OpenAI API key: `OPENAI_API_KEY=your_api_key_here`
3. **Run the analyzer**:
   - **Double-click** the .app file to launch, or
   - Run from Terminal: `open WordPress_Support_Analyzer_vX.X.X.app --args -file="your_data.csv"`

## Usage Examples

### Double-Click Mode (Recommended)
1. Double-click `WordPress_Support_Analyzer_vX.X.X.app`
2. The Terminal will open and prompt you for the input file path
3. Enter the path to your CSV file and press Enter

### Command Line Mode
```bash
# Open with file argument
open WordPress_Support_Analyzer_vX.X.X.app --args -file="path/to/your/data.csv"

# Open with record limit
open WordPress_Support_Analyzer_vX.X.X.app --args -file="data.csv" -limit=1000

# Direct executable access (for advanced users)
./WordPress_Support_Analyzer_vX.X.X.app/Contents/MacOS/WordPress_Support_Analyzer_vX.X.X -file="data.csv"
```

## Requirements

- **macOS 10.13+** (High Sierra or later)
- **No Python installation needed** - this is a standalone application
- **OpenAI API Key** - required for AI analysis (add to .env file)
- **Input CSV file** - containing support data with required columns

## Setup

1. Download the `.app` file
2. Create `.env` file in the same folder:
   ```
   OPENAI_API_KEY=sk-proj-your_api_key_here
   ```
3. Double-click to run!
