# WordPress Support Analyzer GUI - User Guide

## Quick Start

1. **Double-click** the application to launch
2. **Enter your OpenAI API key** (starts with sk-proj-...)
3. **Select your CSV file** using the Browse button
4. **Choose which analyses to run** (all selected by default)
5. **Click "Start Analysis"** and wait for completion

## Features

✨ **Beautiful GUI Interface**
- Easy file selection with drag & drop
- Real-time progress tracking
- Live log display
- Analysis module selection

🔧 **Flexible Analysis Options**
- Core CSAT & Sentiment Analysis
- Data Cleanup & Validation  
- CSAT Prediction Analysis
- Topic Categorization
- Trend Analysis (CSAT, Product Feedback, Goals)
- Optional Visualizations

⚙️ **Advanced Options**
- Configurable record limits for testing
- Cancel running analysis
- Save analysis logs
- Open output folder directly

## Requirements

- **OpenAI API Key**: Get yours at https://platform.openai.com/api-keys
- **CSV Data File**: Your WordPress support data
